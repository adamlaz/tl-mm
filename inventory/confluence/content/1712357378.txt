Release Overview 1.1 High-level Summary of the Release This release includes a quick fix for the Transaction data not showing on the Action Panels. A workaround has been formulated and applied for this issue. The “Transaction Fees (In-Store)” for single day for a single merchant. All of these entires are combined into one “Transaction Fees (In-Store)” lines. Similarly “Transaction Fees (Online)” items are handled. A user will not see multiple entries of Transaction fees In-Store/Online entries for a single day.
